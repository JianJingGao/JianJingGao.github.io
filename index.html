一、环境安装与配置

​ 今天开始学习Golang语言，下面记录每一步学习的节点总结。学习一门编程语言，最基础的就是要搭建其运行环境和熟悉基本语法了，搭建Golang的运行环境则需要先下载相应的二进制文件。

​ Golang是一个由Google开发的开源的编译型的静态语言，他的主要关注点是使得高可用性和可扩展性的web应用开发变得简易。

​ Go语言的优点有：

  • ​ 并发是语言的一部分，编写多线程程序会非常容易。主要倚靠Goroutine和channels机制实现
  • ​ Golang是一种编译型语言。源代码会被编译为二进制机器码，而解释型语言没有这个过程
  • ​ 支持静态链接，所以代码可以静态链接为一个大的二进制文件

安装

1.下载语言包及IDE

​ Golang支持三个主流平台，其有个特性是可以跨平台编译，即A平台编译B平台运行。你可以在 https://golang.org/dl/中下载相应平台的二进制文件。

​ IDE：集成开发环境，使用顺手的IDE可以有效帮助我们提升开发效率，Golang开发中常用的IDE有VS Code、Jetbrains Goland等，目前我选择的是第二个，因为…..顺手啊~前者免费，后者收费的，但是后者易用性和界面布局都让我很喜欢。VS Code还需要另外安装插件。目前我是使用教育认证，取得了一年期的license使用。

​ Jetbrains Goland：点击下载

2.配置环境

Mac OS

在 https://golang.org/dl/ 下载安装程序。双击开始安装并且遵循安装提示，会将 Golang 安装到 /usr/local/go 目录下，同时 /usr/local/go/bin 文件夹也会被添加到 PATH 环境变量中。

Windows

在 https://golang.org/dl/ 下载 MSI 安装程序。双击开始安装并且遵循安装提示，会将 Golang 安装到 C:\Go 目录下，同时 c:\Go\bin 目录也会被添加到你的 PATH 环境变量中。

Linux

在 https://golang.org/dl/ 下载 tar 文件，并解压到 /usr/local。

请添加 /usr/local/go/bin 到 PATH 环境变量中。Go 就已经成功安装在 Linux 上了。

3.配置IDE

​

​ 一路选择next

​ 之后勾选相应软件版本，我选择的是64位并且关联.go文件

​

​ 到安装完成后选择finish即可

​ 打开软件会提示选择购买还是免费使用三十天就看你的选择了。
